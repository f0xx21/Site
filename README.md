# Конвертер валют



Сайт с разделами **Калькулятор** и **Статьи**. Калькулятор конвертирует **EUR**, **USD**, **RUB** и **CNY**, умеет прибавлять процент к сумме.



## Запуск



1. Откройте файл `index.html` в браузере.

2. При первом запуске нужен интернет для загрузки курсов.

3. Курсы кэшируются в браузере на 1 час.



## Разделы



- **Калькулятор** — конвертер валют и расчёт суммы с процентом

- **Статьи** — подразделы с книгами в PDF, просмотр внутри сайта



## Книги



Файлы в папке `books/`:



- `tom-1.pdf` … `tom-6.pdf` — «Основы социологии», Величко

- `ethics.pdf` — «ВП СССР. Об этике и её роли в жизни», Величко

- `5-yazikov-lubvi.pdf` — «5 языков любви», Гэри Чепмен

- `deniken-*.epub` (4 книги) — Эрих фон Дэникен: «Небесные учителя», «История ошибочна», «Послания и знаки из космоса», «Загадочные рисунки Наска»

При проблемах с просмотром через `file://` запустите **`start-server.bat`** и откройте http://localhost:3000. Для EPUB нужен локальный сервер.



## Файлы



- `index.html` — разметка сайта и калькулятора

- `styles.css` — тёмная тема, сайдбар, адаптив

- `nav.js` — переключение разделов

- `books.js` — открытие PDF-книг во встроенном просмотрщике

- `app.js` — логика конвертации и кэширования

- `books/` — PDF-книги
